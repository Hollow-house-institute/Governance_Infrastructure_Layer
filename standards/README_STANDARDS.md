# HHI README Standards

Every repository README should explain:
- what the repository is
- what governance problem it solves
- why the problem matters
- where it fits within the HHI ecosystem
- related repositories
- onboarding flow
- canonical governance references

The objective is ecosystem readability, governance continuity, and institutional coherence.

Policies describe intent.

Runtime behavior reveals reality.

Time turns behavior into infrastructure.

Behavior is the most honest data there is.
