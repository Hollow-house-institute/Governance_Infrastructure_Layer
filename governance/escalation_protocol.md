# Escalation Protocol

## Levels

Level 0 — Normal  
Level 1 — Drift detected  
Level 2 — Escalation risk  
Level 3 — Governance failure  

## Intervention Threshold

- Level 2 → Human-in-the-Loop required  
- Level 3 → Stop Authority triggered  

## Actions

| Level | Action |
|------|--------|
| 1 | Log + monitor |
| 2 | Pause + require review |
| 3 | Stop system |
